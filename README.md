# ArtXpark Corporate Website

ArtXpark는 실물 문화 자산의 진위, 상태, 가치, 자산화를 위한 AI 인프라 기술을 구축하는 Vertical AI + Data Infrastructure 기업입니다.

## 프로젝트 개요

이 프로젝트는 ArtXpark의 공식 기업 웹사이트로, 투자 유치 및 파트너십 구축을 위한 프리미엄 기업 정체성을 전달하도록 설계되었습니다.

## 주요 기능 및 섹션

- **Hero**: ARTENA AI Intelligence Engine의 핵심 가치 제안
- **Market Problem**: 문화 자산 시장의 데이터 인프라 부재 문제 분석
- **ARTENA AI**: 4계층 지능 엔진(Authenticity, Interpretation, Value Signal, Personalization) 상세 구조
- **Product Tiers**: 일반 사용자 및 전문가/기관용 서비스 모델
- **Physical Infrastructure**: Art Mobility Lab 및 무인 스캐닝 인프라(ArtXdrone & ArtXbot)
- **Leadership**: 파운더 및 전략 자문단 프로필

## 기술 스택

- **HTML5**: 시맨틱 웹 구조 적용
- **CSS3**: 프리미엄 다크 테마(Deep Charcoal + Tesla Blue Accent), 그리드 시스템, Glassmorphism
- **정적 배포**: Vercel 및 GitHub Pages에 최적화됨

## 로컬 실행 방법

1. 저장소를 클론합니다.
2. `index.html` 파일을 브라우저에서 엽니다.

## 배포 방법 (Vercel)

1. GitHub 저장소를 Vercel에 연결합니다.
2. 빌드 설정은 기본값(Static Site)을 사용합니다.
3. 자동으로 배포가 완료됩니다.
